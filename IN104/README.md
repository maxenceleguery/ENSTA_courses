# IN104 Project : piano key dectection in language C

## Dependencies

| Librairies |
| ---------- |
| GSL        |
| SDL2       |
| SDL2 - TTF |

* Linux : 
```
$ sudo apt install libgsl-dev
$ sudo apt install libsdl2-dev
$ sudo apt install libsdl2-ttf-dev
```

## Compilation and execution

```
$ make
$ ./output/main
```
